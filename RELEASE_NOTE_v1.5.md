# 🚀 Fast Image Siêu Cấp VIP PRO — Release v1.5.0

**Ngày phát hành:** 04/03/2026
**Nền tảng:** Windows (Portable — không cần cài đặt)
**Kích thước:** ~86 MB

---

## 🎉 Điểm nổi bật của phiên bản này

### 🖼️ Xử lý ảnh tích hợp (Tính năng mới hoàn toàn)
Không cần phần mềm ngoài — giờ bạn có thể **resize, crop và chuyển đổi định dạng ảnh** ngay trong app!

- **Resize**: Thu phóng theo kích thước pixel (W × H) hoặc theo phần trăm
- **Crop**: Cắt theo kích thước hoặc tỷ lệ khung hình tùy chỉnh (vd: `16:9`, `1:1`)
- **Chuyển đổi định dạng**: JPG ↔ PNG ↔ WebP ↔ AVIF
- **Kiểm soát chất lượng**: Tự động hoặc thủ công (1–100)
- **Giữ file gốc**: Tạo file mới có đuôi suffix, file gốc không bị xóa

### 📂 Chọn nhiều file riêng lẻ
- Thêm nút **"Select Files"** — chọn từng file thay vì phải chọn cả thư mục
- Hỗ trợ lọc sẵn: Ảnh thông thường, RAW Camera (CR2, NEF, ARW...), Tất cả file

### 🔢 Vị trí số thứ tự
- Đặt số thứ tự ở **đầu** hoặc **cuối** tên file (Prefix / Suffix)

### 🌗 Theme tự động
- Chế độ **Auto** — tự động theo Dark/Light mode của Windows, đổi ngay khi hệ thống thay đổi

---

## 🔧 Cải tiến quan trọng

- Dropdown menu không còn bị che khuất bởi container — render thẳng vào `<body>`
- Dropdown tự động **lật lên** nếu không đủ chỗ bên dưới
- Preview table hiển thị **kích thước ảnh gốc** (W × H) ngay dưới tên file
- Nút **✕** trên mỗi hàng — xóa từng file khỏi danh sách mà không cần làm lại từ đầu
- Tooltip không còn bị treo trên màn hình

---

## 🐛 Lỗi đã sửa

- Dropdown bị cắt khi container có `overflow: hidden`
- Số thứ tự đặt sai vị trí (prefix/suffix bị nhầm)
- Preview ảnh không refresh khi chuyển tab
- Danh sách file không cập nhật đúng sau khi convert định dạng
